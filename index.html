About Me

Hello World! My name is Victoria Udebiuwa and I am a Software Development student in the CUNY Techworks program at QCC. I began my journey back in January of 2019, in the hopes of getting the tools and learning the skills necessary to become a full-fledged Fullstack Developer. So far in my studies, I've done projects using the markup languages, HTML and CSS, as well as front-end programming language, Javascript. I'm about 3 months into my program so I'm just getting started, but I put this webpage together in order to document my progress. Below in my collection, I've showcased a bit of my work. The first four projects were completed as a part of NYPL's Project Code program and include HTML and CSS, while my last four projects which were done as a part of QCC's CUNY TECH Works Software Development program focus on HTML, CSS, and Javascript. While you're here, feel free to take a look at some of my projects, and/or send me a message!

Reflection

Project One
For this assignment, I was asked to create a basic html site with a main title, 3 paragraphs, 3 images, and at least 2 links. With that in mind, I created a fun "hipster" website. On my own, I added a background color to the body and heading elements because I knew how and wanted to give the page a little style.
Project Two
In this assignment, I had learned a bit more HTML and CSS, and built out a mock travel website. On the HTML side, I included headings, paragraphs, unordered lists, links and images. On the CSS side, I had to customize h1 and h2 elements with font color and alignment, as well as add background color to the body.
Project Three
As I continued to learn more CSS principles, my assignments continued to evolve in terms of complexity. For this particular assignment, I was asked to use the float property to wrap some text next to an image, use the background properties to give the header a background image, and change the font family of headings using google fonts.
Project Four
The focus for this assignment was CSS flexbox. For it, I was asked to recreate a page using different flexbox properties. Now, the original page that I recreated is only available on repl.it, but trust me when I say I nailed it!
Project Five
Beginning with this assignment, Javascript is also incorporated along with HTML and CSS. I was asked to create a fortune teller site that accepts user data and uses it to generate a fortune. I'm a big pokemon fan so I enlisted my favorite pokemon, Eevee, as the "fortune teller." I further personalized my work by adding a second page.
Project Six
For this assignment, I was asked to create a shopping cart that accepts user data and generates a total based on which items and how many of those items are in the shopping cart. I used images and CSS flexbox to style this site.
Project Seven
As I learned more Javascript, the assignments grew more challenging. For this one, I was asked to create a credit card validator that accepts user data and determines whether or not the numbers enter form a valid credit card number. This was a tough one! I decided to have fun with the style and go for a mystery theme. Is your credit card number valid? Let's find out!
Project Eight
This final assignment focused on Youtube APIs. APIs are Application Programming Interfaces. APIs allow you to use documentation from different social media platforms and access features, which you can then apply to your webpage without having to leave your site. I decided to include funny dog blooper vids because honestly, what's better than that?
